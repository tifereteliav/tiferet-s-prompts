# Tiferet's Prompts (פורטפוליו פרומפטים רפואיים ושיווקיים)

מערכת אינטראקטיבית וחדשנית המרכזת ומנגישה פרומפטים מתקדמים לתקשורת רפואית (Medical Communications) ושיווק HealthTech יוקרתי. האתר מעוצב בסגנון Bento Grid מודרני, מבוסס על ערכות צבעים משקמות ומשלב עקרונות לשיפור חוויית משתמש והפחתת עומס קוגניטיבי (Cognitive Load Theory).

## 🚀 תכונות עיקריות
- **פרומפט למצגת רפואית קלינית**: התאמה אישית של מספר השקפים, נושא קליני, קהל יעד ושפת פלט.
- **פרומפט לסרטון סקירה כללית**: פרומפט חזותי יוקרתי המותאם למחוללי וידאו מבוססי בינה מלאכותית (Sora 2, Runway Gen-3).
- **פרומפט למתווה אינפוגרפיקה רפואית**: יצירת אינפוגרפיקות מונגשות וממוקדות-מטופל, המפשטות פרוטוקולים רפואיים.
- **מגירת מידע והסברים קליניים**: לצד כל פרומפט ישנו רקע מדעי מפורט, טבלאות השוואה, הנחיות עיצוב וקוד הפרומפט המערכתי המלא (Master System Prompt) להעתקה מהירה.
- **ממשק משתמש מתקדם**: הדגשת משתנים בזמן אמת, העתקה בלחיצה אחת, ואנימציות כניסה מרהיבות.

## 📂 מבנה הפרויקט
- `index.html` - דף המבנה הראשי (SPA) מותאם לעברית ולכיוון כתיבה RTL.
- `index.css` - עיצוב האתר המשלב Glassmorphic Cards, Bento Grid, ופלטת צבעים מתוחכמת (Transformative Teal).
- `index.js` - הלוגיקה האינטראקטיבית וניהול מצבי המשתמש.
- `prompts_data.js` - בסיס הנתונים המכיל את הפרומפטים, המשתנים ומדריכי הרקע.
- `tiferet.jpeg` - תמונת הפרופיל של תפארת אליאב.
- `logo.png` - לוגו המותג.

## 💻 הרצה מקומית
ניתן להריץ את האתר באמצעות כל שרת סטטי מקומי. למשל באמצעות Node.js:
```bash
npx http-server -p 8000
```
או פשוט לפתוח את קובץ `index.html` ישירות בדפדפן.
